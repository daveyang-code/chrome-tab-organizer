# Privacy Policy

**Last Updated:** March 5, 2025

## Data Collection
This extension, **Tab Organizer**, does not collect, store, or share any user data. It only interacts with the userâ€™s browser tabs to organize them based on their URLs and YouTube video durations. No personal or sensitive information is accessed or transmitted.

## Permissions
The extension requires the following permissions:

- **`tabs`**: To query, move, and organize tabs.
- **`tabGroups`**: To group tabs by their root domain.
- **Access to YouTube pages (`*://www.youtube.com/*`)**: To retrieve video durations for the "Sort YouTube by Length" feature.
